# CloudDisk Frontend

React frontend for CloudDisk cloud storage application.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on `http://localhost:3000` and proxy API requests to `http://localhost:8888`.

## Build

```bash
npm run build
```

## Features

- User authentication (login/register)
- File management (upload, download, delete)
- Folder navigation
- Friend system
- File sharing with friends
